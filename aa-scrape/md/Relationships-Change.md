#### Activity Title
Relationships Change
#### Activity Type
Class Meeting
#### Objective
Recognize that there are different kinds of friendships
#### Themes
![Relationships Icon](http://v5cmservice.secondstep.org/MS3TP_IMAGES/SKILLS/SKILLS_SMALL_IMAGES/relationships-sm.png)Relationships
 
![Values Icon](http://v5cmservice.secondstep.org/MS3TP_IMAGES/SKILLS/SKILLS_SMALL_IMAGES/values-sm.png)Values
 

#### Preparation
1. If possible, seat the class in a circle.
2. Sit where you can participate comfortably in the discussion.

#### Instructions Header
Class Meeting Instructions
#### Warm-Up
Have everyone in the class (including you!) briefly answer the following question: 

**What are some different kinds of friends?**<br/>
            (Best friend. Acquaintance. Someone I play sports with. My neighbor.)
#### Discussion
As a class, discuss one or more of the following prompts, as time allows:


-  **Is it possible to have more than one best friend at the same time?**

-  **How can an acquaintance turn into a best friend?**

-  **When friendships change, do they have to end?**
#### Reflection
Have the class reflect quietly about the following prompt:

**How can having different kinds of friendships be useful?**

Call on students to tell the class their ideas, as time allows.
